з»ҲдәҺеҹәжң¬еҲ°дәҶжңҖеҗҺзҡ„дёҖжӯҘдәҶгҖӮеҫ®дҝЎйӣҶжҲҗдё»иҰҒеҢ…еҗ«дёүдёӘйғЁеҲҶпјҡ

  1. жҺҘе…ҘйӘҢиҜҒпјҢи®©еҫ®дҝЎзЎ®и®ӨдҪ зҡ„жңҚеҠЎеҷЁпјҢе°ұжҳҜдҪ е…¬дј—еҸ·зҡ„жӢҘжңүдәәгҖӮ
  2. жҺҘж”¶еҫ®дҝЎз»ҷдҪ жңҚеҠЎеҷЁзҡ„ж¶ҲжҒҜгҖӮ
  3. еҸ‘йҖҒж¶ҲжҒҜз»ҷеҫ®дҝЎжңҚеҠЎеҷЁпјҢи®©е®ғиҪ¬еҸ‘з»ҷдҪ е…¬дј—еҸ·зҡ„е…іжіЁиҖ…гҖӮ

е…¬дј—еҸ·зҡ„еҹәжң¬ејҖеҸ‘пјҢеҫ®дҝЎзҡ„е®ҳж–№ж–ҮжЎЈгҖҢе…Ҙй—ЁжҢҮеҚ—гҖҚе…¶е®һе·Із»Ҹеҹәжң¬еҢ…еҗ«жүҖжңүеҝ…йЎ»иҰҒзҹҘйҒ“зҡ„зҹҘиҜҶгҖӮдҪҶжҳҜпјҢе®ғзҡ„ж–ҮжЎЈжҳҜд»Ҙ Python дёәдҫӢеӯҗпјҢиҝҳжңүдёҖдәӣжҜ”иҫғзңӢиө·жқҘеҗ“жӯ»дәәзҡ„жөҒзЁӢеӣҫпјҢжҠҖжңҜз»ҶиҠӮпјҢе’ҢдёҖдәӣдёӘдәәеҸ·жІЎжі•ејҖйҖҡзҡ„еҠҹиғҪд»Ӣз»ҚгҖӮжүҖд»ҘпјҢиҝҷзҜҮж•ҷзЁӢжҳҜдҫ§йҮҚдәҺдёӘдәәе…¬дј—еҸ·еҰӮдҪ•з”Ё JavaScript жқҘејҖеҸ‘е®һзҺ°гҖӮ

жҺҘе…ҘйӘҢиҜҒ

жҖҺд№Ҳз”іиҜ·еҫ®дҝЎдёӘдәәе…¬дј—еҸ·пјҢжҲ‘е°ұдёҚиҜҙдәҶгҖӮдёӘдәәеҸӘиғҪз”іиҜ·и®ўйҳ…еҸ·гҖӮ

з”іиҜ·жҲҗеҠҹеҗҺпјҢзҷ»еҪ•пјҢзӮ№еҮ»е·Ұиҫ№иҸңеҚ•ж ҸдёӢйқўзҡ„гҖҢејҖеҸ‘гҖҚ->гҖҢеҹәжң¬й…ҚзҪ®гҖҚпјҢеҸҜд»ҘзңӢеҲ°еҰӮдёӢйЎөйқўпјҡ

е…ҲжҠҠ ејҖеҸ‘иҖ…ID(AppID) е’Ң ејҖеҸ‘иҖ…еҜҶз Ғ(AppSecret) еҲҶеҲ«еӨҚеҲ¶еҮәжқҘпјҢдҝқеӯҳеңЁдёҖиҫ№гҖӮд»ӨзүҢ(Token) жҳҜиҮӘе·ұе®ҡзҡ„дёҚи®©еҲ«дәәзҹҘйҒ“зҡ„еӯ—з¬ҰдёІгҖӮдёӢйқўжҲ‘д»¬жқҘзңӢзңӢзі»з»ҹйҮҢеҰӮдҪ•й…ҚзҪ®пјҢжҺҘе…Ҙеҫ®дҝЎгҖӮ

жү“ејҖе‘Ҫд»ӨиЎҢпјҢиҝӣе…ҘдҪ зҡ„йЎ№зӣ®зӣ®еҪ•пјҢз„¶еҗҺиҫ“е…Ҙд»ҘдёӢе‘Ҫд»Өпјҡ

yo evergrow:wechat

е®ғдјҡжҸҗйҶ’дҪ жҳҜеҗҰиҰҒиҰҶзӣ– package.json ж–Үд»¶пјҢдҪ еҸҜд»Ҙиҫ“е…Ҙ dпјҢжҢү Enter зңӢжңүд»Җд№ҲеҸҳеҠЁгҖӮе…¶е®һеҸӘжҳҜж–°еҠ дёҖдәӣеҝ…иҰҒзҡ„еҢ…пјҢжүҖд»ҘдҪ иҫ“е…Ҙ YпјҢз„¶еҗҺиҰҶзӣ–е°ұеҸҜд»ҘдәҶгҖӮжӯЈеёёзҡ„иҜқпјҢж“ҚдҪңеҗҺдјҡеҰӮдёӢеӣҫйӮЈж ·пјҢж–°з”ҹжҲҗеҮ дёӘеҫ®дҝЎзӣёе…ізҡ„ж–Үд»¶гҖӮ

жү“ејҖ config/wechat/sample.js ж–Үд»¶пјҢеҶ…е®№еҰӮдёӢпјҡ

дҪ иҰҒеҒҡзҡ„жҳҜпјҢеӨҚеҲ¶ sample.js ж–Үд»¶пјҢйҮҚе‘ҪеҗҚдёә production.jsпјҢз„¶еҗҺжҠҠеүҚйқўдҝқеӯҳзҡ„ ејҖеҸ‘иҖ…ID(AppID) е’Ң ејҖеҸ‘иҖ…еҜҶз Ғ(AppSecret) еҲҶеҲ«иөӢеҖјз»ҷ appId е’Ң appSecretгҖӮappToken дёҚз”ЁзҗҶдјҡгҖӮappToken е’Ң appSecret еҸҜдёҚиҰҒи®©еҲ«дәәзҹҘйҒ“пјҒд»ҘжҲ‘зҡ„дёәдҫӢпјҡ

и®ҫзҪ®е®ҢдәҶд»ҘеҗҺпјҢдҪ е°ұеҸҜд»ҘйғЁзҪІеҲ°жңҚеҠЎеҷЁдёҠйқўдәҶгҖӮ

зЎ®е®ҡдҪ зҡ„жңҚеҠЎеҷЁе·Із»ҸеҗҜеҠЁеҗҺпјҢдҪ е°ұеҸҜд»ҘеңЁеҫ®дҝЎгҖҢеҗҜз”ЁгҖҚдҪ зҡ„жңҚеҠЎеҷЁдәҶгҖӮжңҚеҠЎеҷЁзҡ„ URL е°ұеЎ«дҪ зҡ„жңҚеҠЎеҷЁең°еқҖ(жҲ– IP)пјҢеҠ дёҠжҲ‘д»¬еҫ®дҝЎйӣҶжҲҗжЁЎеқ—зҡ„и·Ҝз”ұ /wechatпјҢжҜ”еҰӮпјҡ http://www.mydomain.com/wechat/гҖӮ

жҺҘж”¶ж¶ҲжҒҜ

еҗҜз”ЁеҗҺеӨ§жҰӮиҰҒиҝҮдёүдә”еҲҶй’ҹеҸҜиғҪжүҚдјҡз”ҹж•ҲгҖӮз„¶еҗҺпјҢеҰӮжһңдҪ еҸ‘дёҖж®өж–Үеӯ—з»ҷдҪ зҡ„е…¬дј—еҸ·пјҢдҪ зҡ„жңҚеҠЎеҷЁдјҡжү“еҚ°еҮәжҺҘеҸ—еҲ°зҡ„ж¶ҲжҒҜеҶ…е®№гҖӮ

е…іжіЁе…¬дј—еҸ·зҡ„ж¶ҲжҒҜ

еҸ‘йҖҒж¶ҲжҒҜ

зҹҘйҒ“дәҶдёҠйқўеҮ еӨ§зұ»еһӢзҡ„ж¶ҲжҒҜж јејҸпјҢжҲ‘д»¬е°ұеҸҜд»ҘеҒҡдёҖдәӣдёңиҘҝдәҶгҖӮ

еҫ®дҝЎж¶ҲжҒҜзҡ„жҺҘж”¶е’ҢеӨ„зҗҶпјҢйғҪеңЁ wechat/wechat-controller.js йҮҢйқўзҡ„ wechatMsg ж–№жі•йҮҢеӨ„зҗҶе°ұеҸҜд»ҘдәҶгҖӮе®ғй»ҳи®Өз”ҹжҲҗзҡ„д»Јз ҒжҳҜиҝҷж ·зҡ„пјҡ

е…іжіЁиҮӘеҠЁеӣһеӨҚ

еҰӮжһңиҰҒеңЁз”ЁжҲ·е…іжіЁеҗҺиҮӘеҠЁеӣһеӨҚпјҢжҲ‘д»¬еҸҜд»ҘеңЁдёҠйқўзҡ„ж–№жі•йҮҢеҠ еҰӮдёӢзҡ„д»Јз Ғпјҡ

еӣ дёәеҗҜз”ЁиҮӘе·ұзҡ„жңҚеҠЎеҷЁеҗҺпјҢиҮӘе®ҡд№үиҸңеҚ•еҠҹиғҪе°ұжІЎжңүдәҶпјҢжҲ‘еҸӘиғҪеңЁе…іжіЁиҮӘеҠЁеӣһеӨҚеҠ й“ҫжҺҘеҺ»жҲ‘зҡ„еҺҶеҸІж–Үз« йЎөдәҶпјҲе“ӯпјүгҖӮ

еӣһеӨҚз”ЁжҲ·еҸ‘жқҘзҡ„еӣҫзүҮ

еҰӮжһңжҲ‘д»¬жғізӣҙжҺҘжҠҠз”ЁжҲ·еҸ‘з»ҷжҲ‘д»¬зҡ„еӣҫзүҮеӣһеӨҚеӣһеҺ»пјҢжҲ‘д»¬еҸҜд»Ҙиҝҷж ·еҒҡпјҡ

жңүдәҶиҝҷдәӣеҹәзЎҖпјҢдҪ еҸҜд»ҘжҢүиҮӘе·ұзҡ„йңҖжұӮпјҢжқҘе®һзҺ°е…¶е®ғиҮӘеҠЁеӣһеӨҚзҡ„еҠҹиғҪгҖӮдёӢдёҖз« жҲ‘д»¬еҶҚзңӢеҰӮдҪ•з»“еҗҲд№ӢеүҚзҡ„зі»з»ҹпјҢйҡҸжңәжҠҪеҸ–зі»з»ҹжҺҘж”¶зҡ„е…¶е®ғз”ЁжҲ·еҸ‘жқҘзҡ„еӣҫзүҮпјҢиҖҢдёҚжҳҜиҝ”еӣһиҮӘе·ұеҪ“еүҚеҸ‘жқҘеӣҫзүҮгҖӮ
